A few curated questions from the community. You can update these manually or, later on, connect them to a database or form responses.

How can I safely use AI-generated Dynamo scripts in production?

Start by testing scripts in a copy of your model. Use Arqwell’s prompts to request: “Explain what this script does line by line.” Keep a small library of checked scripts and avoid running anything that touches large sets of elements without filters.

Tags: Dynamo · QA · Best practices

Can Arqwell help me generate Rhino scripts for structure layouts?

Yes. Ask the assistant to place structural elements along a curve, then refine spacing, member type, and grouping. Always review the code and step through with simple geometry first.

Tags: Rhino · Structure · Python

How do we build a model QA checklist using AI?

Use Arqwell to draft a checklist for views, sheets, warnings, and naming conventions. Turn those into prompts and small scripts that check each item. Save your favorite prompts so the QA process is repeatable.

Tags: QA · Revit · Prompts
